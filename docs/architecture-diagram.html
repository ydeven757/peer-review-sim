Peer Review Simulator — Architecture

Streamlit · Ollama / Anthropic / OpenAI · ReportLab PDF

Frontend

  • • Streamlit UI at localhost:8501
  • • Sidebar: venue + persona config
  • • Tabs: paste text / upload file
  • • Score cards, verdicts, accordions
  • • PDF download via st.download_button

Processing Pipeline

  • • paper_loader: DOCX / PDF / text
  • • llm_client: unified Ollama/Anthropic/OpenAI interface
  • • reviewer_engine: generate_review() → ReviewResult
  • • meta_review: synthesize_meta_review() + avg scores
  • • pdf_export: ReportLab A4 PDF

LLM Providers

  • • Ollama: localhost:11434 · no API key · offline
  • • Anthropic: Claude Sonnet 4 · ANTHROPIC_API_KEY
  • • OpenAI: GPT-4o · OPENAI_API_KEY
  • • Auto-detected: Ollama checked first (no key needed)
  • • Unified via app/llm_client.py

Outputs

  • • Inline score cards + reviewer verdicts
  • • Expandable individual review sections
  • • Meta-review synthesis panel
  • • Styled A4 PDF (cover + all sections)
  • • st.session_state persists across interactions
