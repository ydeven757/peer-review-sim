Peer Review Simulator — Architecture

Streamlit app · Anthropic/OpenAI LLM backend · ReportLab PDF export

Frontend

  • • Streamlit UI at localhost:8501
  • • Sidebar: venue + persona config
  • • Tabs: paste text / upload file
  • • Score cards, verdicts, accordions
  • • PDF download button

Processing Pipeline

  • • paper_loader: DOCX / PDF / text
  • • reviewer_engine: LLM calls + JSON parse
  • • meta_review: consensus synthesis
  • • pdf_export: ReportLab A4 PDF
  • • config: 6 venues, 3 personas, 5 dims

LLM Backends

  • • Auto-detects ANTHROPIC_API_KEY
  • • Fallback: OPENAI_API_KEY
  • • Claude Sonnet 4 (Anthropic)
  • • GPT-4o (OpenAI)
  • • temperature=0.2–0.3 · max_tokens=4K

Outputs

  • • Inline score cards + verdicts
  • • Expandable individual reviews
  • • Meta-review synthesis
  • • Styled A4 PDF (cover + all sections)
  • • Download via st.download_button
